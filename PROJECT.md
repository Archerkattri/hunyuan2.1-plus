# hunyuan2.1+

HiCache (Hermite-polynomial velocity forecasting, arXiv:2508.16984) deeply integrated into Hunyuan3D-2.1 (Tencent). A clean implementation of the existing HiCache method: the cached CFG velocity is forecast on skipped sampling steps with a dual-scaled Hermite polynomial basis. The HiCache++ (exponential) sibling is `hunyuan2.1-plus-plus`.
